# EmoteForAll Updated
An creepycats SCP:SL Exiled Plugin that adds an `.emote` Command for Human Roles to do Gingerbread Dances from SCP-3114

Made for `v14.2+` of SCP:SL and `v9.13.2+` of Exiled 

Original plugin can be found here - https://github.com/creepycats/EmoteForAll-SL


## Important changes!!
Previously, the plugin attempted to animate the player's own character directly.
Now: When a player triggers an emote, their character is temporarily hidden and replaced by an SCP-3114 (Skeleton) NPC spawned at the same position. The NPC performs the dance animation, while the real player remains invisible and stationary. If the player moves, the NPC is immediately removed and the player reappears normally.

## Installation
Nothing special, install the DLL like a normal plugin in `Exiled/Plugins`

## Permissions
`EmoteForAll.Emote`
